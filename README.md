# Online Learning Management System (LMS)

This project is a **console-based Online Learning Management System** written in C++. It provides a platform for students, instructors, and admins to manage courses, assignments, quizzes, enrollments, and certificates. The system uses text files for persistent data storage.

## Features

### **User Registration & Authentication**
- **Register** as a Student, Instructor, or Admin.
- **Login** with email and password.
- **Role-based dashboards** after login.

### **Profile Management**
- **View Profile:** See your name, email, and role.
- **Edit Profile:** Update your name, email, or password.If any one entity match then data not change. If you want to edit the profile the previous data and data you enter is not same.

### **Student Functionalities**
- **Enroll in Courses:** Choose from Certificate or Non-Certificate courses.
- **View My Courses:** See all enrolled courses, their status, and progress.
- **Attempt Assignments:** Submit answers for course assignments.
- **Attempt Quizzes:** Take quizzes for enrolled courses.
- **Track Progress:** View completion status for assignments and quizzes.
- **Apply for Certificate:** Apply for a certificate after completing a certificate course.

### **Instructor Functionalities**
- **Upload Assignments:** Create and upload assignments for courses.
- **Upload Quizzes:** Create and upload quizzes for courses.
- **View Course Details:** See enrolled students and their progress in a course.
- **View All Instructors:** List all instructors in the system.

### **Admin Functionalities**
- **Add/Manage Courses:** Add new Certificate or Non-Certificate courses.
- **View All Courses:** List all available courses.
- **View All Students:** List all registered students.
- **View All Instructors:** List all registered instructors.
- **View Enrollment Reports:** See all student enrollments and their statuses.
- **Add New Admin:** Register new admin users.

------------------------------------
## Implementation Details
This project is developed using Object-Oriented Programming (OOP) concepts in C++, including encapsulation, inheritance, and aggregation. It manages users, students, instructors, and admins through a role-based system. Data is stored and retrieved using text files, ensuring simple persistence. The design is modular, reusable, and simulates a basic Learning Management System (LMS).

### Dependencies
The project uses standard C++ libraries. Ensure a compatible compiler environment for compilation and execution.
